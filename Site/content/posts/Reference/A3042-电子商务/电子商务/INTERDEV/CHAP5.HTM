Visual InterDev是为那些想制作高水平的动态网络应用程序的开发者设计的,而不是非程序员。Visual InterDev支持以小组为单位的开发,而且和Microsoft FrontPage97 （一种面向用户和设计者的工具）完全兼容。因此,开发小组和非程序员能在网点上一起工作。
Visual InterDev基于以下的几种关键的设计目标：

1. 集成可视化开发环境。
Visual InterDev 包括了一个综合开发环境，这个开发环境集成了所有编写和配置网络 应用程序的所必须的工具的。

2. 支持编写交互式服务器应用程序。
这个软件对交互式网络应用的支持是Mirosoft Internet Information Server3.0的一个新的特征。交互式服务器主页使编写有高级服务器处理过程的动态网络应用程序更容易,例如：数据库的连接,状态管理，服务器的scripting语言和可复用的服务器组件。Visual InterDev是编写交互式服务器应用程序最好的工具。

图2几乎所有的网络节点都是由程序员和非程序员组成的小组一起制作和维护。既然，小组成员能在同一个网点同步工作,每一个人都能使用适合自己需要的工具，所以联合使用Microsoft FrontPage 和 Microsoft Visual InterDev 是理想的方案.

3.强大而集成的数据库工具
Visual InterDev提供所有他网络开发工具中提供的高级数据库开发功能。
Visual interDev提供很好基于INTRANETS和INTERNET的数据库连接,支持开放数据库关联（ODBC）。它不但包括数据库管理桌面,例如：Micro-soft Access和Microsoft Visual Foxpro, 而且也包括数据库管理系统，例如Micro-soft SQL Server, Oracle, informix 和IBM DB/2。
4.网点管理和内容开发工具
Visual InterDev包括了和Microsoft FrontPage兼容的一套完整的网点管理，包括一套独一无二的用来辅助管理正在在线网点的可视化工具。而Visual InterDev包括一个和网络图象,音效开发工具一样的FrontPage WYSIWYG HTML编辑器。
5. 开放性和可扩充性
Visual InterDev的设计是支持工业标准的，例如：HTML,HTTP,ODBC,ActiveX，COM和Java。因此,它提供了和其他工具和平台的最大程度的兼容性,也提供了用其他类型组件的良好扩展。
